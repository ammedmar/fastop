Computational notes
===================

The repository also includes longer, versioned research notes. They record
the prime-five computations, surface scaling measurements, Sage adapter work,
and the completed example-search program. These notes are deliberately kept
as Markdown source files because they are useful as lightweight research
records; the generated Sphinx site is the package-facing documentation.

* ``../prime-five.md`` records the prime-five validation family.
* ``../symmetric-surfaces.md`` records the genus scaling program.
* ``../roadmap.md`` records completed milestones and release work.
* ``../package-api.md`` gives a compact Markdown version of this guide for
  readers browsing the source tree.
